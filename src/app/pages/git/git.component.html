Git

  • To configure the user name and email address

    git config --global user.name "full_name"

    git config --global user.email "email_address"

  • To tell Git to verify SSL or not (for HTTP and HTTPS)

    git config --global https.sslVerify false | true

    git config --global http.sslVerify | true

  • To set and configure HTTP/HTTPS proxy

    git config --global https.proxy http://user:pass@yourproxy:port

    git config --global http.proxy http://user:pass@yourproxy:port

    git config --global --unset https.proxy

    git config --global --unset http.proxy

  • To set/add remote address of the Git repository

    git remote set-url origin $GIT_REPOSITORY_URL

    git remote add origin $GIT_REPOSITORY_URL

  • To count the commits in a date range in your current branch

    git rev-list --count HEAD --since="Dec 3 2015" --before="Jan 3 2016"

  • If you want the count for all branches in one go use --all additionally

    git rev-list --count --since="Dec 3 2015" --before="Jan 3 2016" --all

  • If you want to exclude merge-commits, use option --no-merges

    git rev-list --count --since="Dec 3 2015" --before="Jan 3 2016" --all --no-merges

  • To set the default text editor for all Git operations

    git config --global core.editor emacs

  • To set a template for commit messages

    git config --global commit.template ~/.gitmessage.txt

  • To set the default color of the GUI

    git config --global color.ui false | auto | always | true

  • To configure CRLF option for files

    git config --global core.autocrlf true | input | false

  • To configure checking of Git object sanity

    git config --system receive.fsckObjects true

  • To tell Git to refuse force-pushes

    git config --system receive.denyNonFastForwards true
