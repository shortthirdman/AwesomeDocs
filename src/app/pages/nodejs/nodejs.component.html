Node.js

  • To set the NPM registry server in Windows

    npm config set registry https://registry.npmjs.org/

  • To set shell for npm run-scripts in Windows (x64)

    npm config set script-shell "C:\\Program Files\\git\\bin\\bash.exe"

  • To set shell for npm run-scripts in Windows (x86)

    npm config set script-shell "C:\\Program Files (x86)\\git\\bin\\bash.exe"

  • To unset shell for npm run-scripts in Windows

    npm config delete script-shell

  • Increase Node memory to 1GB

    node --max-old-space-size=1024 index.js

  • Increase Node memory to 2GB

    node --max-old-space-size=2048 index.js

  • Increase Node memory to 3GB

    node --max-old-space-size=3072 index.js

  • Increase Node memory to 4GB

    node --max-old-space-size=4096 index.js

  • Increase Node memory to 5GB

  • Increase Node memory to 6GB
